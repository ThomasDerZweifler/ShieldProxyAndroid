Shield Proxy For Android
========================

Running the app:
1) Download the APK from https://github.com/cgutman/ShieldProxyAndroid/releases
2) Install the APK on your Shield
3) Ensure the Shield proxy is running on your PC and your router is configured properly (see readme for the Windows relay)
   The latest version can be downloaded from https://github.com/cgutman/ShieldProxyWindows/releases
3) Run the Shield Proxy app on your Shield
4) Fill in the externally accessible IP address or DNS name of your PC
5) Tap start to start the service. If errors are reported,
   check that the IP address or name is specified correctly and that the router is properly forwarding
   the correct TCP and UDP ports to your PC.

Getting the code:
- The Shield Streaming Proxy for Windows code is available at https://github.com/cgutman/ShieldProxyWindows
- The Shield Streaming Proxy for Android code is available at https://github.com/cgutman/ShieldProxyAndroid

Building the app:
1) The code supplied includes an Eclipse project file which should build with the most recent version of Eclipse
